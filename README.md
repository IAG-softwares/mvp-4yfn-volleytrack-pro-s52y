# VolleyTrack Pro

A comprehensive volleyball club management platform for managing players, scheduling matches, tracking performance statistics, and streamlining team operations.

## Quick Start

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

The frontend will be available at http://localhost:5173
The backend API will be available at http://localhost:3001

## Project Structure

```
├── frontend/          # React + Vite frontend
```

## Features

Player management with detailed profiles and statistics, match scheduling and result tracking, player authentication and self-service portal, performance analytics dashboard, attendance tracking, position management, team roster organization, match history and statistics, automated notifications for matches and events, mobile-responsive design for on-the-go access, data export capabilities for reporting, and role-based access control for different user types.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Tailwind CSS
